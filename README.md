# Brain_Tumor_ImageClassification_UsingDensnet121
This project focuses on detecting and classifying brain tumors from MRI images using a deep learning model based on DenseNet121. The system classifies images into four categories: Glioma, Meningioma, Pituitary tumor, and No Tumor.

The system includes preprocessing techniques like masking and normalization, statistical evaluation, and a web-based deployment for real-time prediction.

Objectives
Develop a CNN model using DenseNet121
Perform accurate brain tumor classification
Apply image masking to enhance tumor region detection
Evaluate model using statistical metrics
Deploy the model using a web application

Dataset Information
Dataset Path: C:\Brain_Tumor\Epic and CSCR hospital Dataset
Total Classes: 4
Glioma
Meningioma
Pituitary
No Tumor

Model Details
Base Model: DenseNet121 (Pretrained on ImageNet)
Transfer Learning Applied
Output Layer: Softmax (4 classes)
Optimizer: Adam
Loss Function: Categorical Crossentropy
